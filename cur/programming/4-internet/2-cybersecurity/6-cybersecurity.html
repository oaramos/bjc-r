¿Qué puedes hacer?

En esta página, aprenderá formas de proteger sus datos y su computadora.

No debería ser responsabilidad del individuo garantizar su seguridad en línea, al igual que no es su responsabilidad hacer una inspección de seguridad de cada vagón del metro antes de abordar. Sin una regulación adecuada, no hay soluciones perfectas que usted, como individuo, pueda usar para asegurarse de que nunca será victimizado. Sin embargo, hay cosas que puede hacer que le ayudarán:

  1. Ejecute software actualizado. El sistema operativo Windows 98 no se vendió después de 2000 y no se admitió después de 2006. Pero todavía hay computadoras que ejecutan este sistema obsoleto. sistema, incluidos muchos en el Departamento de Defensa de los EE. UU. siguen usando sistemas obsoletos, es porque dependen de un software de aplicación que solo se ejecuta en el sistema antiguo.
  2. Utilice la autenticación para proteger los dispositivos y la información del acceso no autorizado. Por ejemplo:

      Una contraseña segura es una que es difícil de adivinar.

    1. Use contraseñas seguras. Necesita una contraseña distinta para cada sitio que utilice. La única buena solución es usar un administrador de contraseñas, un programa que genera una contraseña aleatoria para cada sitio. Solo recuerda una contraseña, la del administrador de contraseñas. Cuida a los demás por ti.
    2. Habilite la autenticación multifactor (como la autenticación de dos factores) para que solo pueda acceder a su dispositivo o cuenta después de ingresar información específica (generalmente una es una contraseña y la otra requiere otro de sus dispositivos o cuentas o el otro usa algo que detecta tu cuerpo como un lector de huellas dactilares). Puede parecer una molestia, pero cada paso agregado al progreso de inicio de sesión agrega otra capa de seguridad.
  3. No haga clic en enlaces en sitios web o especialmente en correos electrónicos, sin antes verificar dos veces que la URL real en el enlace es la que espera. (¿A dónde lo envía realmente este enlace a http://google.com?) Si una computadora o cuenta de alguien sabe que ha sido comprometida, puede recibir correos electrónicos que parecen provenir de ellos y que contienen enlaces maliciosos que podrían comprometer su sistema o cuenta. Como sabe, es importante mantener su software actualizado, pero un truco común es una ventana emergente que le pide que descargue una actualización. No haga clic en ellos a menos que esté seguro de que realmente es una actualización legítima. Lo mejor es ir a la sección "actualizaciones" de la aplicación o del sistema operativo para obtener actualizaciones.
  4. No utilice software incompleto. Si la publicidad dice que el programa le dará dinero, cosas gratis, pornografía o trampas para videojuegos, es muy probable que se trate de malware. ¡Una categoría particularmente astuta es el software antivirus falso! Consulte las reseñas en las revistas (y vuelva a verificar el enlace desde el que planea descargar) para asegurarse de que está obteniendo lo que realmente desea.
  5. Revise periódicamente la configuración de seguridad de los sitios web, las aplicaciones y los programas que utiliza. Puede controlar los permisos para recopilar y compartir su información.
  6. Instala software antivirus y antimalware. Hay versiones gratuitas de varios paquetes populares disponibles. Lea las reseñas para encontrar el software que se ajuste a sus necesidades y presupuesto.
  7. No te conectes a Wi-Fi inseguro. Idealmente, las redes Wi-Fi deberían usar seguridad WPA2 o WPA3 (WiFi Protected Access), pero WPA es mejor que ninguna seguridad.
